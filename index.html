Política de Privacidade

Última atualização: 16/02/2026

A presente Política de Privacidade descreve como a aplicação ClaraZap recolhe, utiliza e protege as informações dos utilizadores.

1. Informações Recolhidas

Podemos recolher as seguintes informações:

  • Nome e endereço de email
  • Número de telefone
  • Informações necessárias para autenticação (ex: login via Google ou GitHub)
  • Dados técnicos como endereço IP e tipo de navegador

2. Utilização das Informações

As informações recolhidas são utilizadas para:

  • Fornecer e melhorar os nossos serviços
  • Autenticar utilizadores
  • Comunicar com os utilizadores quando necessário
  • Cumprir obrigações legais

3. Partilha de Dados

Não vendemos nem alugamos dados pessoais a terceiros. Os dados podem ser partilhados apenas quando necessário para:

  • Cumprimento de obrigações legais
  • Prestação de serviços técnicos essenciais (ex: infraestrutura de alojamento, APIs oficiais)

4. Segurança

Adotamos medidas técnicas e organizacionais adequadas para proteger os dados pessoais contra acesso não autorizado, alteração ou divulgação.

5. Retenção de Dados

Os dados pessoais são mantidos apenas pelo tempo necessário para cumprir as finalidades descritas nesta política, salvo quando a lei exigir um período de retenção superior.

6. Direitos do Utilizador

Nos termos da legislação aplicável (incluindo o RGPD, quando aplicável), o utilizador pode solicitar:

  • Acesso aos seus dados
  • Correção de dados incorretos
  • Eliminação dos seus dados
  • Limitação ou oposição ao tratamento

7. Contacto

Para qualquer questão relacionada com esta Política de Privacidade, pode contactar-nos através do email: samu.plantaarvores@gmail.com
